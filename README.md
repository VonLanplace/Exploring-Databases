# Exploring Databases 🗄️

Welcome to my database repository! This project serves as a centralized hub for my practice scripts, lab exercises, and personal projects as I learn **SQL Server** this semester. 

The goal of this repository is to document my journey from writing basic queries to mastering complex database management and optimization.

---

## 🛠️ Tech Stack
* **Database Engine:** Microsoft SQL Server
* **Management Tool:** [DBeaver Community Edition](https://dbeaver.io/)
* **Language:** T-SQL (Transact-SQL)

---

## 📚 Learning Objectives
This semester, I am focusing on the following core areas:
* **DDL (Data Definition Language):** Creating and altering tables, schemas, and constraints.
* **DML (Data Manipulation Language):** Mastering `INSERT`, `UPDATE`, and `DELETE`.
* **Complex Queries:** Joining multiple tables, using subqueries, and CTEs (Common Table Expressions).
* **Programmability:** Writing Stored Procedures, Functions, and Triggers.
* **Optimization:** Understanding execution plans and basic indexing.

## 🚀 How to Use These Scripts
1.  **Clone the Repo:** 
    ```bash
    git clone https://github.com/your-username/Exploring-Databases.git
    ```
2.  **Environment:** Ensure you have **SQL Server Express** or **Developer Edition** installed.
3.  **Execution:** Open the `.sql` files in **SSMS**, connect to your local instance, and execute the scripts.
